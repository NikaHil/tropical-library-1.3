# tropical-library-1.3

Добавлены ачивки. Теперь есть 5 достижений. Создали новую таблицу в базе данных и файл achievements.php, где прописана логика выдачи наград. В файле database.php подключили этот модуль. Страница статистики stats.php полностью обновлена — теперь там показываются все ачивки, шкала прогресса и сколько осталось до следующей. На главной странице index.php добавили виджет с полученными наградами и всплывающие уведомления, когда получаешь новую ачивку. Файлы add.php и edit.php теперь проверяют ачивки при добавлении или изменении книги. Для пользователя это значит, что учёт книг стал похож на игру: ты видишь свой прогресс, получаешь награды за цели и хочешь читать больше и разнообразнее.
